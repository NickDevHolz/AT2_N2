# Simulador de Playlist em C

Este repositório contém um programa em C que simula uma playlist de músicas. O programa lê um arquivo de texto (`musicas.txt`) contendo artistas e suas respectivas músicas, armazena esses dados em uma lista circular duplamente encadeada e permite ao usuário interagir com a playlist através de um menu interativo.

## Funcionalidades

O programa oferece as seguintes funcionalidades através de um menu interativo:

1. **Exibição da playlist pela ordem de cadastro:** Mostra todas as músicas na ordem em que foram cadastradas.
2. **Exibição da playlist ordenada pelo nome das músicas:** Mostra todas as músicas ordenadas alfabeticamente pelo nome.
3. **Inserção de novas músicas:** Permite ao usuário adicionar novas músicas à playlist.
4. **Remoção de uma música:** Permite ao usuário remover uma música da playlist.
5. **Busca por uma música:** Permite ao usuário buscar uma música específica na playlist.
6. **Avançar para próxima música:** Avança para a próxima música na playlist.
7. **Retornar a música anterior:** Retorna para a música anterior na playlist.

As funcionalidades de inserção e remoção também atualizam o arquivo `musicas.txt`.

## Arquivo de Entrada

O arquivo `musicas.txt` contém os registros das músicas e artistas, separados por ponto e vírgula (`;`). Exemplo:


Artista X;Música 3 Artista Y;Música 42


## Como Compilar e Executar

Para compilar e executar o programa, siga estas etapas:

1. Clone o repositório para sua máquina local.
2. Navegue até o diretório do projeto.
3. Compile o programa com o comando `gcc -o simulador_playlist simulador_playlist.c`.
4. Execute o programa compilado com o comando `./simulador_playlist`.
5. Interaja com o menu interativo para gerenciar sua playlist.

## Requisitos

- GCC (GNU Compiler Collection)
